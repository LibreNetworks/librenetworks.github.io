Who We Are

Libre Networks is a volunteer-driven open-source group building tools and libraries designed to improve and extend the capabilities of the MyOwnFreeHost (MOFH) platform. Our projects support developers by offering clean abstractions, API wrappers, modules, and utilities for both backend and frontend integration.

Featured Project

mofh-fossbilling-module

An unofficial FOSSBilling module that seamlessly integrates MyOwnFreeHost account provisioning, domain management, and automation directly into FOSSBilling.

  • Fully open-source (Apache 2.0)
  • Automated account creation for MOFH
  • API-driven integration
  • Simple to install and extend
